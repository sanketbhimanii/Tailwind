color :- In CSS, a color can be specified by using a predefined color name
        
        RGB:- Red Green Blue
        
        RGBA :- Red Green Blue Alpha(for transparency)
        
        HEX Value :- using HEX desimal value 
                     ex.#ff6347
        
        HSL Value :- In CSS, a color can be specified using hue, saturation, and lightness.
                     ex:hsl(0, 100%, 50%)

accent-color :- The accent-color property specifies the accent color for user-interface controls like: <input type="checkbox">, 
               <input type="radio">, <input type="range"> and <progress>.

align-content :- The align-content property specifies how flex lines are distributed along the cross axis in a flexbox container.
              -This property works only when the “flex-wrap:” property is set to wrap.

align-iteams :- The align-items property specifies the default alignment for items inside a flexbox or grid container.
              -This property works even when the “flex-wrap:” property is not set to wrap.

align-self :- The align-self property specifies the alignment in the block direction for the selected item inside a flexbox or grid container.
